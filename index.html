Como Usar

  1. Para converter de decimal para binário: digite um valor no campo "Digite um número decimal" e clique em "Converter para Binário".
  2. Para converter de binário para decimal: digite um valor no campo "Digite um número binário" e clique em "Converter para Decimal".
  3. Se quiser ver um exemplo, clique em "Gerar Número Binário Aleatório" para gerar um número binário e, em seguida, clique em "Converter para Decimal".
  4. O resultado das conversões aparecerá na caixa de resultado acima.
